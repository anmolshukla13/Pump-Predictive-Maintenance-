## Predictive Maintenance of Pumps & Valves

This project applies machine learning to predict equipment failure in oil and gas operations using real-time sensor data.

### Key Features
- Sensor-based failure prediction
- Random Forest classification model
- Explainable feature importance
- Industry-relevant metrics

### Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

### Business Impact
- Early fault detection
- Reduced unplanned downtime
- Improved maintenance planning
